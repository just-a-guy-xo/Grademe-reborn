Assignment name  : ft_print_reverse_alphabet
Expected files   : ft_print_reverse_alphabet.c
Allowed functions: write
--------------------------------------------------------------------------------

Write a function that displays the alphabet in lowercase on a single line in
reverse order followed by a newline.

Your function must be declared as follows:

void	ft_print_reverse_alphabet(void);

Example:

$> ./ft_print_reverse_alphabet | cat -e
zyxwvutsrqponmlkjihgfedcba$
