Assignment name  : ft_recursive_factorial
Expected files   : ft_recursive_factorial.c
Allowed functions: None
--------------------------------------------------------------------------------

Write a function that returns the factorial of the number given as a
parameter.

If the argument is less than 0, the function should return 0.

The result of the factorial will be stored in an int; if the result cannot be
stored within an int, the return value is undefined.

Your function must be declared as follows:

int	ft_recursive_factorial(int nb);

Example:

$> ./ft_recursive_factorial 5 | cat -e
120$
