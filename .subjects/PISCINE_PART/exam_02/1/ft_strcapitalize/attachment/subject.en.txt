Assignment name  : ft_strcapitalize
Expected files   : ft_strcapitalize.c
Allowed functions: None
--------------------------------------------------------------------------------

Write a function that capitalizes the first letter of each word and sets the
rest of the word to lowercase.

A word is a string of alphanumeric characters. All other characters should be
considered as delimiters.

The function must be declared as follows:

char	*ft_strcapitalize(char *str);
